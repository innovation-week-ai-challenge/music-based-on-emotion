# AI Challenge - play music based on emotions

## ✪ About this project

- This project is an AI-based challenge where we determine the emotion of a given picture and play a generated song using AI that empowers the emotion.
- 😀 Happy
- 😥 Sad
- 😠 Angry
- 😵 Confused
- 🤢 Disgusted
- 😦 Surprised
- 😐 Calm
- ❓ Unknown
- 😨 Fear
